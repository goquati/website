Datenschutz

Quati setzt den Schutz Ihrer Privatsphäre und die Sicherheit Ihrer persönlichen Daten an erste Stelle. Diese Datenschutzrichtlinie soll Ihnen helfen zu verstehen, wie wir Ihre persönlichen Daten erfassen und verarbeiten, wenn Sie unsere Webseite nutzen oder uns kontaktieren.


Wir glauben, dass Sie das Recht haben, Ihre persönlichen Daten zu kontrollieren. Deshalb haben wir die verschiedenen Rechte, die Sie bezüglich Ihrer persönlichen Daten haben, dargelegt, einschließlich Ihres Rechts, Widerspruch gegen bestimmte Nutzungen einzulegen, sowie Ihr Recht auf Zugriff, Aktualisierung oder Löschung Ihrer Daten.

1. Definitionen

  • „Quati" oder „Wir": bezieht sich auf die Quati GmbH, eine deutsche Gesellschaft eingetragen beim Registergericht Augsburg unter der Register Nummer (aktuell Unbekannt), mit ihrem Sitz in der Augsburger Str. 11, 86551 Aichach, Deutschland.
  • „Nutzer", „Kunde" oder „Sie": bezieht sich auf jede Person, die unsere Dienstleistungen abonniert, darauf zugreift oder sie nutzt.
  • „Datenschutzrichtlinie": bezieht sich auf dieses Dokument, das die Verarbeitungsaktivitäten beschreibt, die von Quati als Datenverantwortliche durchgeführt werden. Diese Datenschutzrichtlinie umfasst die Verarbeitungsaktivitäten in Bezug auf die Nutzung unserer Dienstleistungen.
  • „Verarbeitung": bezieht sich auf jede Operation in Bezug auf Ihre persönlichen Daten (zum Beispiel: Sammlung, Nutzung, Zugriff, Übertragung, Löschung, etc.).
  • „Personenbezogene Daten" oder „Benutzerdaten": bezieht sich auf alle Daten, die direkt oder indirekt mit Ihnen in Verbindung stehen.
  • „Datenverantwortlicher": Bezieht sich auf die Person, die Entscheidungen über Ihre persönlichen Daten trifft – z.B., welche Daten gesammelt, gespeichert und wie lange sie aufbewahrt werden.

2. Wer ist für den Datenschutz verantwortlich?

Quati ist verantwortlich für die Verarbeitung Ihrer persönlichen Daten. Gemäß Artikel 37 des Bundesdatenschutzgesetzes ist Quati nicht verpflichtet, einen Datenschutzbeauftragten zu benennen. Aufgrund der Lage von Quati ist die zuständige Datenschutzaufsichtsbehörde das Bayerische Landesamt für Datenschutzaufsicht. Aktuelle Kontaktdaten entnehmen Sie bitte deren Webseite: https://www.lda.bayern.de.

3. Welche personenbezogenen Daten sammeln wir und warum?

Gemäß dem Prinzip der Datenminimierung streben wir an, die gesammelten Daten auf ein Minimum zu reduzieren. Allerdings werden personenbezogene Daten gesammelt, wenn Sie die Webseite nutzen und wenn Sie Quati kontaktieren.

3.1 Welche persönlichen Daten werden bei Besuch der Website gesammelt?

Wenn Sie unsere Website besuchen, werden Ihre IP-Adresse, Browserinformationen, Zeitstempel und vergleichbare Informationen als Teil der Serverprotokolle gespeichert. Wir verarbeiten Serverprotokolle auf der Grundlage unseres berechtigten Interesses, die Sicherheit und ordnungsgemäße Funktion unserer Website zu gewährleisten. Diese Daten werden nicht für andere Zwecke verarbeitet.

3.2 Was passiert mit meinen Daten, wenn ich Sie schriftlich kontaktiere?

Wenn Sie uns schriftlich per Post oder E-Mail kontaktieren, verarbeiten wir Ihre personenbezogenen Daten auf Grund unseres berechtigten Interesses, auf Ihre Anfragen zu antworten. Die Daten, die Sie uns bei der Kontaktaufnahme zur Verfügung stellen, bleiben bei uns, bis Sie um Löschung bitten, Ihre Einwilligung zur Speicherung widerrufen oder der Zweck für die Datenspeicherung entfällt, z. B. nachdem Ihre Anfrage bearbeitet wurde. Gesetzliche Bestimmungen bleiben unberührt.

4. An wen geben wir Ihre persönlichen Daten weiter?

  • GitHub Pages: Unsere Website wird auf GitHub Pages gehostet, einem Dienst der GitHub, Inc. Informationen zur Verarbeitung Ihrer Daten durch GitHub finden Sie unter GitHub Datenschutzerklärung.
  • Microsoft Outlook: Für die E-Mail-Kommunikation setzen wir Microsoft Outlook ein. Weitere Informationen zur Datenverarbeitung durch Microsoft erhalten Sie in deren Datenschutzerklärung.

5. Welche Rechte haben Sie als Benutzer?

Unter dem Datenschutzgesetz haben Sie als Benutzer folgende Rechte:

  • Das Recht auf Information
  • Das Recht auf Berichtigung und Löschung
  • Das Recht auf Einschränkung der Verarbeitung
  • Das Recht auf Widerspruch gegen die Verarbeitung
  • Das Recht auf Datenübertragbarkeit
